# Google広告検索語句レポート自動診断ツール

## 1. プロジェクトの目的

Google広告の検索語句レポートCSVをアップロードすると、検索クエリごとの日次・週次変化を自動分析し、コンバージョン悪化、CPA悪化、無駄費用、勝ちクエリ、機会損失を検知するWebアプリを構築する。

本ツールの本質は、広告運用者が以下を即座に判断できる状態を作ることである。

- どのクエリがCV減少の原因になっているか
- どのクエリが費用を使っているのにCVしていないか
- どのクエリが伸びており、強化対象か
- どのクエリを除外候補・要確認候補にすべきか
- 前期間と比較して、どの指標が悪化原因なのか

単なるCSVビューアではなく、広告改善アクションを出す診断ツールとする。

5.5 CVRとCPAの扱い

CVRとCPAはCSVに存在していても、原則として内部で再計算する。

CVR = conversions / clicks
CPA = cost / conversions

ただし、conversionsが0の場合、CPAは数値ではなくnullとして扱う。

8. 母数不足ルール

以下のいずれかに該当する場合は、原則として強い判定を避け、「監視」に寄せる。

今期間費用が3000円未満
今期間表示回数が100未満
前期間と今期間の合計クリック数が20未満

ただし、今期間費用が10000円以上かつCVが0の場合は、母数不足でも無駄クエリ候補として扱う。

9. クエリ分類ロジック
9.1 分類一覧
分類	意味	推奨アクション
勝ちクエリ	CV増加またはCPA改善が見られる	予算・入札・広告文強化
悪化クエリ	CV減少かつCPA悪化	LP・広告文・検索意図確認
無駄クエリ	費用増加かつCVなし	除外候補または要確認
機会損失クエリ	クリック・費用は増加しているがCVが伸びていない	訴求・LP・クエリ意図確認
新規クエリ	前期間になく今期間に出現	要観察
消滅クエリ	前期間にあったが今期間に出現しない	配信減少・需要減少確認
監視クエリ	母数不足または変化小	保留
9.2 勝ちクエリ条件

以下のいずれかを満たす場合。

今期間CV > 前期間CV かつ CPAが悪化していない
今期間CVが2以上 かつ CPAが前期間より改善
今期間CVが前期間比で増加し、費用増加率よりCV増加率が高い
9.3 悪化クエリ条件

以下を満たす場合。

前期間CVが1以上
今期間CV < 前期間CV
今期間CPAが前期間CPAより悪化、または今期間CVが0
母数不足ルールに該当しない
9.4 無駄クエリ条件

以下を満たす場合。

今期間CV = 0
今期間費用 >= 10000円
今期間クリック数 >= 20

または

今期間費用が前期間より増加
今期間CV = 0
前期間も今期間もCV = 0
合計費用が15000円以上
9.5 機会損失クエリ条件

以下を満たす場合。

今期間クリック数 > 前期間クリック数
今期間費用 > 前期間費用
今期間CV <= 前期間CV
CVRが悪化している

これは除外ではなく、訴求・LP・検索意図確認の候補とする。

9.6 新規クエリ条件
前期間データなし
今期間データあり

新規クエリは原則として「新規クエリ」とし、今期間費用が10000円以上かつCVが0の場合のみ「無駄クエリ候補」とする。

9.7 消滅クエリ条件
前期間データあり
今期間データなし

前期間CVがあった場合は「消滅した勝ちクエリ」としてアラート表示する。

9.8 監視クエリ条件

以下のいずれか。

母数不足
変化が小さい
CVも費用も小さい
判断材料が不足している
10. 除外候補の安全装置

以下に該当するクエリは、無駄クエリ条件を満たしていても即除外候補にはせず、「要確認」にする。

ブランド名を含む
指名検索と推定される
商品名を含む
商標名を含む
症状名や悩み名など高意図ワードを含む
過去にCV実績がある
CVまでの検討期間が長い商材である可能性がある

ブランド名・商品名・除外禁止ワードは、ユーザーが画面上で任意登録できるようにする。

MVPではテキスト入力欄で除外禁止ワードをカンマ区切り登録できればよい。

11. 優先度スコア
11.1 目的

分析結果を単に分類するだけでなく、広告運用者がどこから見るべきかを明確にする。

11.2 スコア計算

各クエリにpriorityScoreを付与する。

基本式：

priorityScore =
  CV減少インパクト
  + 費用悪化インパクト
  + CPA悪化インパクト
  + 機会損失インパクト
11.3 各インパクト
CV減少インパクト = max(0, 前期間CV - 今期間CV) × 10000

費用悪化インパクト = max(0, 今期間費用 - 前期間費用)

CPA悪化インパクト =
  前期間CPAと今期間CPAが存在する場合
  max(0, 今期間CPA - 前期間CPA)

機会損失インパクト =
  今期間クリック増加 かつ CV増加なしの場合
  max(0, 今期間費用 - 前期間費用) + max(0, 前期間CVR - 今期間CVR) × 10000
11.4 ソート優先順位

ランキング表の初期ソートは以下。

priorityScore降順
CV減少数降順
費用増加額降順
今期間費用降順

13. 推奨アクション生成ルール

LLMを使わなくても、ルールベースでコメントを生成する。

13.1 コメント例

勝ちクエリ：

CVが増加しており、CPAも悪化していません。予算・入札強化、広告文への反映を検討してください。

悪化クエリ：

前期間よりCVが減少し、CPAも悪化しています。検索意図、広告文、LP訴求のズレを確認してください。

無駄クエリ：

一定以上の費用を使用していますが、CVが発生していません。除外候補として確認してください。

機会損失クエリ：

クリックと費用は増えていますが、CVが伸びていません。流入意図とLPの一致度を確認してください。

新規クエリ：

今期間から新しく発生したクエリです。費用とCVの推移を数日監視してください。

消滅クエリ：

前期間には配信実績がありましたが、今期間は出現していません。配信量低下や需要変化を確認してください。

20. このツールの思想

このツールは、広告運用者の判断を自動化するものではない。

目的は、広告運用者が見るべきクエリを最短で発見し、判断の質と速度を上げることである。

したがって、除外・強化・改善の判断は断定せず、必ず数値根拠と注意点を併記する。
